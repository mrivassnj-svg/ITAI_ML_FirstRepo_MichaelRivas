## Getting Started with GitHub & Your First Repository

##  Objective
This repository is part of the **Classic Machine Learning** course assignment designed to introduce students to GitHub. The goal is to gain hands-on experience with creating repositories, collaborating with teammates, and using branches and pull requests.

By completing this assignment, students will:
- Create a GitHub account
- Set up their first public repository
- Upload and modify files
- Collaborate with teammates using GitHub branches and pull requests
- Learn how GitHub will be used throughout the course for submitting and sharing machine learning projects

---

##  Why This Matters
GitHub is a critical tool used by data scientists, AI engineers, and software developers. It allows teams to collaborate efficiently, track changes, and showcase work to potential employers. Becoming comfortable with GitHub is an essential foundational skill for this course and future projects.
